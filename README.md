### A PyTorch implementation of [An End-to-End Khmer Optical Character Recognition using Sequence-to-Sequence with Attention](https://arxiv.org/abs/2106.10875)


### Usage

```shell
# generate dataset
python generate.py

# start the training
python train.py

# inference
python infer.py
```
